LLumine Books – Fullstack Online Bookstore

Lumine Books is a modern Fullstack Web Application built with React, Node.js, and Express.
The goal is to create a fast, elegant, and intuitive online bookstore with a clean UI and a solid backend architecture.

This project is currently in development and will evolve to include authentication, cart management, an admin dashboard, and full database integration.

 Technologies Used:
Frontend:
-React.js
-React Router

Backend:
-Node.js
-Express.js

Project Architecture:
/frontend   → React (User Interface)
/backend    → Node.js + Express (REST API)
